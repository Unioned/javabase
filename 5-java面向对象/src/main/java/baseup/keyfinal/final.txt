final修饰符
    方法  表明该方法是最终方法,不能被重写
    类    表明该类是最终类,不能被继承
    变量  叫做常量,只能被赋值一次
        final修饰基本数据类型,记录的值不能发生改变。
        final修饰引用数据类型,记录的地址值不能改变,但是其内部的属性值可以发生改变
            例：一个数组用final修饰后数组中的值仍然可以改变,但是不能被其他数组赋值(覆盖地址)了。

native关键字
    native 的本意是原生的,本地的。
    native 是一个修饰符,只可以用来修饰方法。
    native 修饰的方法,看不见方法体。
    native 修饰的方法,不是用 Java 语言实现的，而是调用了底层 C/C++ 的代码，这些代码为 .dll 文件，让 Java来执行。
