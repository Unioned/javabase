static表示静态，是Java中的一个修饰符，可以修饰成员方法，成员变量

    static修饰的成员变量，叫做静态变量
    参考类：class StudentMain   class Student   static内存图: gif通过浏览器打开
        特点:
            ● 所有对象共享该变量的属性(new出的对象依然有这个变量上一次的赋值)
            ● 不属于对象,属于类
            ● 静态变量初始化是随着类模板的加载而加载的,优先于对象在栈内存中出现
        调用方式:
            ● 类名调用(推荐)
            ● 对象名调用

    static修饰的成员方法，叫做静态方法
    参考类：class ArrayUtil   class UtilTest
        特点:
            ● 多用在测试类和工具类中。
            ● Javabean类中很少会用。
        调用方式:
            ● 类名调用(推荐)
            ● 对象名调用(在工具类中会私有化构造方法,不让外界创建其对象)。

各种类作用简介:
    Javabean类:用来描述一类事物的类。 比如，Student, Teacher, Dog, Cat等。
    测试类:用来检查其他类是否书写正确，带有main方法的类，是程序的入口。
    工具类:不是用来描述一类事物的，而是帮我们做一些事情的类。

static注意事项
    参考类：class StaticStudent   class StaticTest
    ● 静态方法中是没有this关键字。
    ● 静态方法中只能直接访问静态变量和静态方法(没有对应的this自然不能访问非静态变量,可以通过new对应对象调用非静态方法)。
    ● 非静态方法可以访问静态变量或者静态方法，也可以访问非静态的成员变量和非静态的成员方法,
        即非静态可以访问所有。
