数组(容器)
    数组是一组数据类型相同的数据的组合,可以将数据进行统一管理
    数组本身为引用数据类型,数组内储存的类型可以为基本类型,也可以是引用类型
    数组的特点:
        1.数组本身是一个引用数据类型
        2.数组是存储在堆内存中的一串连续地址
        3.数组初始化是必须指定长度
        4.堆内存中的数组长度一旦确定,不能再次发生改变
        5.栈内存的变量中储存的是数组的地址引用
        6.数组内部存储的数据可以是基本数据类型,也可以是引用数据类型


    1.数组定义(声明)
        数据类型[]  数组名字;
        例：
         int[]      x;
         char[]     y;
         String[]   n;

    2.数组的赋值(初始化)
        静态初始化   有长度,有元素
        例:
            int[] array = new int[] {数组数据(逗号分隔)};
            int[] array = {数组数据(逗号分隔)};
        动态初始化   有长度,元素为默认值
        例:                      (整型--0;浮点型--0.0;字符型--'0'(无法输出);布尔型--false;引用数据类型--null)
            int[] array = new int[5];
        异常:
            int[] array = new int[-1];
            NegativeArraySizeException 运行时异常 数据长度不合法

    3.数组元素的访问
        通过数组元素中的索引(index)进行访问
        索引范围[0-->数组长度-1]
        数组索引超出范围会出现运行时异常 ArrayIndexOutOfBoundsException

    4.数组的遍历
        通过循环的方式来访问数组的每一个元素
            for(;;){}
        JDK1.5 新特性 增强for循环 forEach
            for(定义的变量(用于接收数组中的每一个元素):要遍历的数组){}

        正常for循环和新版本for循环区别:
            1.正常for
                三个必要条件  index索引   起始位置   终止条件
                    存值和取值都可以    写法相对麻烦
            2.加强for
                两个条件    用来取值的变量     用来遍历的数组
                写法相对容易
                只能取值    不能存值
                没有索引    不能确定到某一个具体元素

    *5.基本数据类型和引用数据类型的内存结构的区别
        所有的变量空间都存储在栈内存
        变量空间可以存储基本数据类型,也可以存储引用数据类型
        如果变量空间存储的是基本数据类型,存储的是值,一个改变另一个不会改变
        如果变量空间存储的是引用数据类型,存储的是地址(引用),变量地址的值改变,另一个跟着改变

多维数组

    1. 定义/声明
        里面存储的类型[] 数组名字
        int[][] array;

    2. 初始化
            静态---有长度 有元素
                int[][]  a = {{1,2},{3,4,5,6},{7,8,9}};
            冬天---有长度 元素为默认值
                int[][] b = new int[4][3];

    3.元素访问
        通过元素在数组位置---index
        array[i][j]
        i控制大数组中小数组的位置
        j控制小数组中元素的位置

    4.遍历轮询
        嵌套for(原版和加强for都可)
        例:  class code.TestArray.java

    5.引用类型(二维数组)在内存中的存储结构
        语句  int[][] array = new int[3][2];
              array[0][0] = 10;
              array[0][1] = 20;
              内存图: 见多维数组内存1.png

        语句  int[][] array = new int[3][];
              内存图: 见多维数组内存2.png
              此时进行调用array[0][0] = 10;这种语句将会抛出NullPointException异常
