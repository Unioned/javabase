类型转换
    类型转换问题：
        1.同种数据类型可以直接进行赋值操作
            int a = 1 ;
            int b = a ;

            float x = 3.4;
            float y = x;
        2.数据类型不同的空间之间的赋值--->类型转换问题
            同种大数据类型才能进行直接转换
            基本类型---基本类型之间   可以直接转换（自动/强制）
            引用类型---引用类型之间   可以直接转换（自动/强制 向上/下转型）
            基本类型---引用类型之间   不可以直接转换（间接---包装类/封装类）
        3.保证大数据类型相同的前提下
            基本类型---基本类型
                小数据类型相同
                    都是整型或者都是浮点数
                        byte a = 1;
                        int b = 1;
                        float x = 3.4F;
                        double y = x;
                        （小空间->大空间）可以直接进行自动转换
                        int a = 1;
                        byte b = (byte)a;   转换只留下最后八个bit位
                        double x = 3.4;
                        float y = (float)x;
                        （大空间->小空间）需要进行强制类型转换
                总结：
                    大空间变量可以直接存储小空间变量数据
                    小空间变量不可以直接存大空间变量数据（需要强制类型转换）,但转换过程中数值范围超过边界可能会有损失。
                小数据类型不同：
                    浮点型--整型：
                        两个比较精确程度,浮点数精确程度更高,可以直接存放整数,反制需要进行类型强制转换
                    整型--字符型：
                        每个字符对应一个Unicode编码 a--97
                        char x = 'a'; int y = x; //自动转换 y--97
                        int x = 97 ; char y = (char)x;//需要强制类型转换
                    布尔类型：
                        不同于C语言,不能与其他类型发生转换
